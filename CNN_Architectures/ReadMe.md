## CNN Architectures

This directory contains implementations and study notes on classical Convolutional Neural Network (CNN) architectures. The objective is to understand how CNNs have evolved over time, the design principles behind each architecture, and their applications in computer vision.

### Architectures Covered

- LeNet
- AlexNet
- VGG-16
- GoogLeNet (Inception v1)
- Custom CNN Architecture

### Overview

Each implementation explores the motivation behind the architecture, its key building blocks, and its contribution to the evolution of deep learning for image classification. Where applicable, experiments include model implementation, training, and performance evaluation.

The custom CNN architecture was developed as part of an experimental image classification task, demonstrating the application of CNN design principles to a practical problem.

### Tools & Frameworks

- Python
- TensorFlow / Keras
- PyTorch (where applicable)
- Jupyter Notebook

### Future Additions

This directory will continue to expand with implementations of modern CNN architectures, performance comparisons, and transfer learning experiments.

### Note
The purpose of these notebooks is to demonstrate the implementation of classic CNN architectures on a dataset. The focus is on understanding the architecture and its implementation rather than achieving the highest possible performance. Hyperparameter tuning, model optimization, and advanced training techniques have not been applied. For a more comprehensive CNN implementation that includes preprocessing, regularization, optimization, and model improvement techniques, please refer to the **CNN for Pneumonia Detection from Chest X-Rays** notebook in this repository.
