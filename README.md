## Deep Learning

This repository is a growing collection of deep learning concepts, architectures, and implementations. It is intended to build a strong understanding of how modern deep learning models are designed, trained, and applied to solve real-world problems.

### Overview

The repository focuses on studying the evolution of deep learning architectures, understanding their underlying principles, and implementing them for practical applications across different domains.

### Current Content

#### Deep Learning Architectures
- Fundamental neural network architectures
- Feedforward Neural Networks (FFNN/ANN)
- CNN Architectures (LeNet, AlexNet, VGG16, GoogleNet)
- Architecture summaries and implementation notes
